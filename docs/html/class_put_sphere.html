função para colocar uma esfera.

Variáveis para o centro em x (xs) (xs = x da esfera), centro em y (ys), centro em z (zs), raio, r para o tom de vermelho, g para o tom de verde, b para o tom de azul e a para a transparência alpha. As cores misturadas podem formar outras cores. Cada variável r, g, b e a podem ir de 0.0 a 1.0 ou de 0 a 255.
