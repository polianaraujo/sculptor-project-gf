Detailed Description

A struct Voxel é uma estrutura para a classe Sculptor, que esta será chamada caso não tenha sido definida em outro arquivo.

Nela estão as variáveis das cores, tranparência e uma booleana.

Member Data Documentation

◆ a

float Voxel::a

A variável de transparência pode ir de 0 (totalmente sem cor) até 1 (totalmente opaca) ou de 0 a 255.

◆ r

float Voxel::r

Variáveis para as cores r (vermelho), g (verde) e b (azul), que misturadas podem formar outras cores. Cada variável pode ir de 0.0 a 1.0 ou de 0 a 255.

◆ show

bool Voxel::show

A booleana show vai definir se o voxel[][][] vai ser mostrado (true) ou não (false).
